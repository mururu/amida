# Amida

A Generator Amidakuji

## Installation

Install it yourself as:

    $ gem install amida

## Usage

    $ amida --width 8 --height 15

Result:

    A   B   C   D
    |   |---|   |
    |---|   |---|
    |   |---|   |
    |   |   |   |
    |   |---|   |
    |---|   |---|
    |---|   |   |
    |   |   |---|
           !!!

Default width is 5 and height is 10.
